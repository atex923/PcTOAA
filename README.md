# PcTOAA

KOHAKU 預算書轉換容器。

## 目前正式版

- `PsTOAA_V4_5_4.py`
- `PsTOAA_V4_5_4.pyw`

目前正式版：V4.5.4

視窗標題：`KOHAKU預算書轉換容器V4.5.4`

匯出預設檔名：`AA3118_工程編號_YYYYMMDDHHMMSS`

## 使用方式

開啟命令列執行：

```bash
python3 PsTOAA_V4_5_4.py
```

Windows 可使用 `.pyw` 啟動無 console 視窗版：

```bash
python3 PsTOAA_V4_5_4.pyw
```

## 歷史版本

舊版程式檔保留於 `history/versions/`。

版本整理紀錄請參考 `history/README.md`。

## 版本進版規則

推送新版號前，根目錄只保留目前正式版的 `.py` 與 `.pyw`。

舊版 `.py` 與 `.pyw` 必須移到 `history/versions/`，避免 A 電腦與 B 電腦同步時誤用舊版或互相覆蓋。
