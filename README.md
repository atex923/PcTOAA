# PcTOAA

KOHAKU 預算書轉換容器。

- `PsTOAA_V4_5_2.py`
- `PsTOAA_V4_5_2.pyw`

目前正式版：V4.5.2

視窗標題：`KOHAKU預算書轉換容器V4.5.2`

輸出預設檔名：`AA3118_工程編號_YYYYMMDDHHMMSS`

## 使用方式

建議在一般桌面環境使用：

```bash
python3 PsTOAA_V4_5_2.py
```

若要使用無終端視窗版本，可直接開啟：

```bash
python3 PsTOAA_V4_5_2.pyw
```
