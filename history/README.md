# PsTOAA 程式生成歷程

目前正式版：`PsTOAA_V4_5_2.py` / `PsTOAA_V4_5_2.pyw`

本資料夾保存本次開發過程中產生的版本快照。根目錄只追蹤目前正式版；舊版與中間版本集中保存在 `history/versions/`，用於追溯需求變更與程式生成歷程。

## 版本摘要

- `PsTOAA_V4_1_3.py`
  - 以既有預算分析工具為基礎，完成組合總表、輸出總表、刪除單複價、整理備註等名稱整理。

- `PsTOAA_V4_2_a.py` 到 `PsTOAA_V4_2_c.py`
  - 調整單價分析表讀取規則。
  - B 欄遇到人工/機具區段時略過，直到 A 欄重新有資料才恢復判讀。
  - B 欄遇到 `工作項目:` 標記列時略過該列。

- `PsTOAA_V4_2_d.py` 到 `PsTOAA_V4_2_e.py`
  - 新增目前分頁搜尋功能。
  - 新增項目名稱、備註超過 25 字的檢查與米橘色底標示。

- `PsTOAA_V4_2_f.py` 到 `PsTOAA_V4_2_g.py`
  - 分析時在組合總表與輸出總表插入項目 `0` 的總工程經費列。
  - 重新計算時彙總第 2 層工項複價，填入項目 `0` 的單價與複價。

- `PsTOAA_V4_2_h.py` 到 `PsTOAA_V4_2_k.py`
  - 右上金額區補齊項目代碼。
  - 新增 `06工程管理費` 級距試算。
  - 新增 `0A3公共藝術費`，顯示預算總表複價與組合總表項目 `01` 複價乘 1% 的試算值。
  - 調整 `檢查字數` 按鈕位置到 `整理備註` 右側。

- `PsTOAA_V4_2_1.py`
  - 將 `V4.2.a` 到 `V4.2.k` 的修正整合為數字版號。
  - 整理程式檔頭說明與版本紀錄。

- `PsTOAA_V4_2_2.py`
  - 匯出預設檔名改為 `AA3118_` 加上西元年月日時分秒。

- `PsTOAA_V4_2_3.py` 到 `PsTOAA_V4_2_4.py`
  - 修正長檔名遮蔽按鈕問題。
  - 修正 `06工程管理費` 試算基礎，並將試算值顯示到小數後三位。
  - 調整上半部部分間距。

- `PsTOAA_V4_2_5.py` 到 `PsTOAA_V4_2_7.py`
  - 上半部新增粉藍色功能區並重排操作按鈕。
  - 新增往後回復編輯功能。
  - 移除匯出 XLS，只保留匯出 XLSM。
  - 將左側與中間功能按鈕統一成立體樣式，並加大上半部與分頁切換區間距。

- `PsTOAA_V4_2_7.py`
  - 前一版正式版。

- `PsTOAA_V4_3_0.py`
  - 視窗最底部新增標語 `Imagination is more important than knowledge.`。
  - 前一版正式版。

- `PsTOAA_V4_3_1.py`
  - 組合總表數量欄顯示到小數後第三位。
  - 重新計算時將右上試算的 `012`、`0A1`、`0A2`、`0A3`、`06` 寫回組合總表對應工項單價，並以單價乘數量更新複價。
  - 右上金額區 `06工程管理費` 移到 `0A1空氣污染防制費` 上方。
  - 前一版正式版。

- `PsTOAA_V4_3_2.py`
  - 右上金額區 `0B營業稅` 移到 `012承包商利潤及工程保險費` 下方。
  - 分析時若下半部已有資料，先詢問是否全部清除並重新載入。
  - 右上金額區全列分成原始資料與重新計算結果兩欄，原始欄保留分析讀入值。
  - 前一版正式版。

- `PsTOAA_V4_3_3.py`
  - 讀入資料時，`單價` 欄統一顯示到小數後第三位。
  - 視窗與上方欄位、功能區、右上金額區改為可隨視窗縮放。
  - 前一版正式版。

- `PsTOAA_V4_3_4.py`
  - 右上金額區 `012` 右欄公式改為 `011×10%`。
  - 右上金額區 `06`、`0A1`、`0A2`、`0A3` 右欄計算結果無條件取到整數位。
  - 前一版正式版。

- `PsTOAA_V4_3_5.py`
  - 右上金額區右欄顯示格式統一為 `#,##0.000`。
  - 右上金額區總經費改為固定加總 `01+02+03+04+05+06+0A+0B`。
  - 前一版正式版。

- `PsTOAA_V4_3_6.py`
  - 右上金額區總經費改為從下半部組合總表讀取 `01+02+03+04+05+06+0A+0B` 各自複價加總。
  - 上半部新增鎖定回填工項欄位，預設鎖定 `012`、`06`、`0A1`、`0A2`、`0A3`、`0A4`、`0A5`。
  - 鎖定工項重新計算時保留手動單價，直接以手動單價乘數量更新複價。
  - 右上 `0A1` 到 `0A6` 支援來源工項與比例設定；`0A4` 到 `0A6` 在下半部存在時自動顯示。
  - 前一版正式版。

- `PsTOAA_V4_3_7.py`
  - 鎖定回填新增啟用勾選框，勾選時才會略過右上回填計算。
  - 右上金額區加寬，避免長數字被欄框遮住。
  - 右上 `0A` 項下動態列擴充到 10 項，支援到 `0AA`。
  - 右上 `0A` 列比例欄為 0 或空白時，最右欄直接讀取下半部對應工項複價。
  - 前一版正式版。

- `PsTOAA_V4_3_8.py`
  - 修正 `0A5` 等 0A 動態列在直接讀複價時，被鎖定顯示邏輯覆蓋成讀單價的問題。
  - 前一版正式版。

- `PsTOAA_V4_3_9.py`
  - 第四分頁 `0A` 項下工項單價改為讀取第二分頁對應工項單價。
  - 重新計算時先處理鎖定回填欄位內的工項；勾選鎖定時保留手動單價並把複價顯示到右上對應欄位。
  - 未勾選鎖定時，右上運算結果會回填到下半部對應工項單價，再計算複價。
  - 鎖定/回填工項處理後，再重新彙總 `01`、`011`、`06`、`0B` 與總經費等上層金額。
  - 前一版正式版。

- `PsTOAA_V4_4_0.py`
  - 第四分頁 `0A1` 到 `0AA` 工項的數量與單價改為讀取第二分頁對應工項資料。
  - 移除左上部鎖定回填欄位、勾選框、編輯紀錄欄位與重新計算中的鎖定分支。
  - 重新計算時右上運算結果一律回填到下半部對應工項單價，再計算複價並彙總上層工項。
  - 回溯前版本。

- `PsTOAA_V4_3_10.py`
  - 版號依指定回歸 `V4.3.10`。
  - 保留 `0A1` 到 `0AA` 工項的數量與單價讀取第二分頁對應工項資料。
  - 重新計算時不再覆蓋下半部既有單價，讓手動修改的單價可以直接參與複價計算。
  - 前一版正式版。

- `PsTOAA_V4_3_11.py`
  - 修正 `0A1` 到 `0AA` 有子項時，重新計算階層彙總覆蓋手動單價的問題。
  - 重新計算時 `0A1` 到 `0AA` 保留目前單價，只用該單價乘數量更新複價。
  - 左上半部欄框加上黑框。
  - 前一版正式版。

- `PsTOAA_V4_3_12.py`
  - 右上半部 `0A1` 到 `0AA` 的重新計算結果改為讀取第四分頁目前複價。
  - 避免右上來源工項/比例公式顯示舊數字，與手動修改後的複價不一致。
  - 前一版正式版。

- `PsTOAA_V4_3_13.py`
  - 執行重新計算與計算工項複價前，強制提交第四分頁目前正在編輯的儲存格。
  - 將第四分頁畫面上的最新文字同步回資料後再計算，避免剛手動修改的單價尚未提交而沿用舊值。
  - 前一版正式版。

- `PsTOAA_V4_3_14.py`
  - 右上 `0A1` 到 `0AA` 最右欄依來源工項與比例規則計算。
  - 來源與比例都有填時，以來源工項複價乘比例；來源空白且比例為 1 時，以該工項單價乘數量；比例為 0 時不計算。
  - 左上與左上中間功能區欄框、按鈕高度固定為約 1.2 倍字高。
  - 右上字體與欄框高度縮小為原來約 80%。
  - 前一版正式版。

- `PsTOAA_V4_3_15.py`
  - 修正第四分頁隱藏階層或只顯示部分資料後，重新計算會用畫面列號覆蓋完整資料列的問題。
  - 手動修改 `0A1` 到 `0AA` 單價後，重新計算會用項目代碼對回原資料列，保留目前單價並以數量重算複價。
  - 指定工項若暫時讀不到數量或單價，重新計算時略過該列，不再把既有複價清成空白。
  - 前一版正式版。

- `PsTOAA_V4_3_16.py`
  - 上半部左側操作區、中間功能區、右上金額區改為可隨視窗比例縮放。
  - 初始視窗大小依目前螢幕可用範圍設定，避免低解析度或不同縮放比例下被遮蔽。
  - 降低上半部欄框與右上金額欄的硬性最小寬度，讓三個區塊在窄視窗時可一起收縮。
  - 前一版正式版。

- `PsTOAA_V4_2.py`
  - 依指定將目前正式版號調整為 `V4.2`。
  - 保留 `V4.3.16` 已完成的上半部自適應縮放與既有計算修正。
  - 將根目錄與來源資料夾內舊版程式檔集中移入 `history/versions/`，根目錄只保留目前正式版。
  - 前一版正式版。

- `PsTOAA_V4_4.pyw`
  - 由 `PsTOAA_V4_4_Codex_Transfer.zip` 匯入主程式並進版為目前正式版。
  - 保留壓縮檔附帶的 V4.4 功能：上半部隱藏/顯示開關文字固定，避免版面跳動。
  - 修正主程式檔頭與預設 log 檔名，使檔名、視窗版號與紀錄檔名一致。
  - 前一版正式版。

- `PsTOAA_V4_4_1.py` / `PsTOAA_V4_4_1.pyw`
  - 第四分頁新增 `整理備註2` 功能按鈕。
  - 檢查第四分頁 `備註` 欄內英文字母開頭、後續由英文與數字組成且包含數字的字串。
  - 符合規則的備註片段以藍色顯示，資料內容維持純文字，不影響編輯與匯出。
  - 前一版正式版。

- `PsTOAA_V4_4_2.py` / `PsTOAA_V4_4_2.pyw`

  - 英文字母開頭且含數字、或數字開頭且含英文字母的英數混合字串，都會以藍色顯示。
  - 前一版正式版。

- `PsTOAA_V4_4_3.py` / `PsTOAA_V4_4_3.pyw`

  - 左側刪除後，備註中剩餘符合規則的英數混合字串仍以藍色顯示。
  - 整理程式檔頭版本紀錄，將重複與中間修正版說明整併為系列摘要。
  - 前一版正式版。

- `PsTOAA_V4_4_4.py` / `PsTOAA_V4_4_4.pyw`

  - `備註整理` 會依序執行分隔符整理、左側英數特殊字串刪除與剩餘英數混合字串藍色標示。

- `PsTOAA_V4_4_5.py` / `PsTOAA_V4_4_5.pyw`

- `PsTOAA_V4_4_6.py` / `PsTOAA_V4_4_6.pyw`
  - 將 `刪除行` 移到左半邊 `組合總表` 區的第一個按鈕。

- `PsTOAA_V4_5_0.py` / `PsTOAA_V4_5_0.pyw`

## 版號整理規則

- 英文尾碼版號轉成數字尾碼時，整合該輪英文尾碼的修正說明。
- 第 3 碼接續編號；第 2 碼是否進版由使用者指定。
