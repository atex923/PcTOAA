# PsTOAA 程式生成歷程

目前正式版：`PsTOAA_V4_2_2.py`

本資料夾保存本次開發過程中產生的版本快照。根目錄保留正式版；`history/versions/` 內的檔案用於追溯需求變更與中間版本。

## 版本摘要

- `PsTOAA_V4_1_3.py`
  - 以既有預算分析工具為基礎，完成組合總表、輸出總表、刪除單複價、整理備註等名稱整理。

- `PsTOAA_V4_2_a.py` 到 `PsTOAA_V4_2_c.py`
  - 調整單價分析表讀取規則。
  - B 欄遇到人工/機具區段時略過，直到 A 欄重新有資料才恢復判讀。
  - B 欄遇到 `工作項目:` 標記列時略過該列。

- `PsTOAA_V4_2_d.py` 到 `PsTOAA_V4_2_e.py`
  - 新增目前分頁搜尋功能。
  - 新增項目名稱、備註超過 25 字的檢查與米橘色底標示。

- `PsTOAA_V4_2_f.py` 到 `PsTOAA_V4_2_g.py`
  - 分析時在組合總表與輸出總表插入項目 `0` 的總工程經費列。
  - 重新計算時彙總第 2 層工項複價，填入項目 `0` 的單價與複價。

- `PsTOAA_V4_2_h.py` 到 `PsTOAA_V4_2_k.py`
  - 右上金額區補齊項目代碼。
  - 新增 `06工程管理費` 級距試算。
  - 新增 `0A3公共藝術費`，顯示預算總表複價與組合總表項目 `01` 複價乘 1% 的試算值。
  - 調整 `檢查字數` 按鈕位置到 `整理備註` 右側。

- `PsTOAA_V4_2_1.py`
  - 將 `V4.2.a` 到 `V4.2.k` 的修正整合為數字版號。
  - 整理程式檔頭說明與版本紀錄。

- `PsTOAA_V4_2_2.py`
  - 目前正式版。
  - 匯出預設檔名改為 `AA3118_` 加上西元年月日時分秒。

- `PsTOAA_V4_2_3.py`
  - 曾產生的中間版本，匯出命名規則後續回溯修正至 `V4.2.2`。

## 版號整理規則

- 英文尾碼版號轉成數字尾碼時，整合該輪英文尾碼的修正說明。
- 第 2 碼進版時，也整合前一個第 2 碼底下的第 3 碼修正說明。
